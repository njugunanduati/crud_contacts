# crud_contacts
A simple django applications that demonstrates crud usng django class based views 

#Getting started

1.) Clone the Project

git clone https://github.com/njugunanduati/crud_contacts.git

2.) cd into the project

 cd project_name
 
3.)Create a virtual environment

  python3 -m venv cenv 
-- cenv is the name of my virtual enviroment. You can name yours whichever

4.) Add the dependancies

pip install -r requirements.txt

5.) Make the migrations and migrate

python manage.py makemigrations

python manage.py migrate

6.) run the project

python manage.py runserver
